# Frontal Harmony Analyzer\n\nAn advanced AI-powered web application for analyzing facial harmony using cutting-edge computer vision technology.\n\n## Features\n\n- **AI-Powered Face Detection**: MediaPipe FaceMesh landmark detection (468-point model)\n- **Advanced Measurements**: 29+ facial harmony measurements across 5 categories\n- **Gaussian Scoring**: Mathematically accurate scoring with percentile analysis\n- **Symmetry Analysis**: Bilateral comparison and asymmetry detection\n- **Beautiful Dashboard**: Premium, modern UI with real-time visualizations\n- **Responsive Design**: Mobile-first, fully responsive interface\n- **Privacy First**: Local processing, no data stored\n\n## Quick Start\n\n### Prerequisites\n\n- Node.js 18+ \n- npm or yarn\n\n### Installation\n\n```bash\n# Clone repository\ngit clone https://github.com/13jmason-stack/Harmony.git\ncd Harmony\n\n# Install dependencies\nnpm install\n\n# Download MediaPipe models\n# Visit: https://github.com/google/mediapipe/releases/tag/v0.10.0\n# Place model files in public/wasm/\n\n# Start development server\nnpm run dev\n```\n\nOpen [http://localhost:3000](http://localhost:3000) to begin.\n\n## Tech Stack\n\n- **Framework**: Next.js 15 with App Router\n- **Language**: TypeScript\n- **Styling**: TailwindCSS with dark-mode\n- **State**: Zustand\n- **Animations**: Framer Motion\n- **Charts**: Recharts\n- **Face Detection**: MediaPipe FaceMesh\n- **Deployment**: Vercel\n\n## Project Structure\n\n```\nsrc/\n├── app/                 # Next.js app routes\n├── components/          # React components\n├── features/\n│   ├── faceDetection/   # MediaPipe integration\n│   ├── measurements/    # Calculation engine\n│   └── rendering/       # Canvas visualization\n├── hooks/               # Custom React hooks\n├── lib/\n│   ├── measurements.ts  # 29 measurements\n│   └── scoring.ts       # Gaussian scoring\n├── store/               # Zustand state\n├── types/               # TypeScript definitions\n└── utils/               # Geometry & utilities\n```\n\n## Measurements\n\n### Eye Harmony (9)\n- Intercanthal distance ratios\n- Interpupillary distance\n- Eyebrow positioning\n- Cantal tilt angles\n\n### Facial Frame (5)\n- Bizygomatic width\n- Gonial width\n- Cheekbone prominence\n- Forehead proportion\n\n### Midface (4)\n- Nose width ratios\n- Nasolabial angle\n- Philtrum length\n- Midface height\n\n### Lower Third (4)\n- Mouth width\n- Chin height\n- Jaw definition\n- Lower third proportion\n\n### Global Proportions (4)\n- Facial thirds balance\n- Face shape index\n- Vertical/horizontal symmetry\n- Distribution analysis\n\n## Scoring System\n\nUses mathematically accurate Gaussian distribution:\n\n```\nrawScore = 100 × exp(-0.5 × ((value - ideal) / tolerance)²)\nweightedScore = rawScore × weight\npercentile = inverse_normal_CDF(score)\n```\n\n## API Documentation\n\n### Face Detection\n\n```typescript\nconst { detectFace, processFaceImage } = useFaceDetection();\n\n// Detect landmarks from image\nconst mesh = await detectFace(imageElement);\n\n// Process file with measurements\nconst result = await processFaceImage(file);\n```\n\n### Analysis\n\n```typescript\nconst { generateReport } = useAnalysis();\n\n// Generate complete analysis report\nconst report = await generateReport(measurements, faceMesh, imageUrl);\n```\n\n## Contributing\n\nContributions are welcome!
